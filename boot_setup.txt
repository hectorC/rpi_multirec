RPi MultiRec boot splash setup for Waveshare 1.3inch LCD HAT on Ubuntu Server

Goal
Show a boot screen on the LCD during boot, keep it visible until rpi_multirec starts, then stop the splash and start the recorder app.

Assumptions
Project path on Pi is /home/hcenteno/rpi_multirec
Recorder binary is /home/hcenteno/rpi_multirec/build/rpi_multirec
Main recorder service is /etc/systemd/system/rpi_multirec.service
Running as root service (needed for SPI/GPIO/I2C access)

Step 1: Install Python runtime deps for splash
sudo apt update
sudo apt install -y python3 python3-pil python3-spidev python3-gpiozero python3-numpy
sudo apt install -y python3-lgpio

Step 2: Verify Waveshare LCD python files are available
This setup uses the python example files you copied into the project at:
/home/hcenteno/rpi_multirec/python

Test import:
PYTHONPATH=/home/hcenteno/rpi_multirec/python python3 -c "import ST7789; print('ST7789 module ok')"

If step 2 fails, install the Waveshare python library first using the wiki package or your local Waveshare examples, then re-run step 2.

Step 3: Create boot splash python script
sudo tee /usr/local/bin/rpi_hat_boot_screen.py > /dev/null <<'PYEOF'
#!/usr/bin/env python3
import signal
import sys
import time

sys.path.append("/home/hcenteno/rpi_multirec/python")
from PIL import Image, ImageDraw, ImageFont
import ST7789

running = True

def stop_handler(signum, frame):
    global running
    running = False

signal.signal(signal.SIGTERM, stop_handler)
signal.signal(signal.SIGINT, stop_handler)

try:
    disp = ST7789.ST7789()
    disp.Init()
    disp.clear()
    disp.bl_DutyCycle(50)
except Exception as e:
    print(f"Boot splash init failed: {e}", file=sys.stderr)
    sys.exit(1)

font_big = ImageFont.load_default()
font_small = ImageFont.load_default()

def draw_frame(dot_count):
    img = Image.new("RGB", (240, 240), "black")
    d = ImageDraw.Draw(img)
    d.text((20, 72), "RPi MULTIREC", fill=(255, 255, 255), font=font_big)
    d.text((20, 108), "Booting system", fill=(0, 220, 220), font=font_small)
    d.text((20, 132), "Please wait" + "." * dot_count, fill=(255, 220, 0), font=font_small)
    d.text((20, 200), "v1 startup", fill=(120, 120, 120), font=font_small)
    disp.ShowImage(img.rotate(270))

dot = 0
while running:
    draw_frame(dot)
    dot = (dot + 1) % 4
    time.sleep(0.4)

try:
    disp.module_exit()
except Exception:
    pass
sys.exit(0)
PYEOF
sudo chmod +x /usr/local/bin/rpi_hat_boot_screen.py

Step 4: Create splash systemd service
sudo tee /etc/systemd/system/rpi_hat_boot_screen.service > /dev/null <<'EOF'
[Unit]
Description=RPi HAT Boot Splash
DefaultDependencies=no
After=local-fs.target
Before=multi-user.target rpi_multirec.service
Conflicts=shutdown.target

[Service]
Type=simple
ExecStart=/usr/bin/python3 /usr/local/bin/rpi_hat_boot_screen.py
Restart=always
RestartSec=1
User=root
Group=root

[Install]
WantedBy=multi-user.target
EOF

Step 5: Update recorder service so it stops splash before launch
sudo tee /etc/systemd/system/rpi_multirec.service > /dev/null <<'EOF'
[Unit]
Description=RPi Multi Recorder
After=systemd-udev-settle.service sound.target rpi_hat_boot_screen.service
Wants=systemd-udev-settle.service rpi_hat_boot_screen.service

[Service]
Type=simple
WorkingDirectory=/home/hcenteno/rpi_multirec
ExecStartPre=/bin/systemctl stop rpi_hat_boot_screen.service
ExecStart=/home/hcenteno/rpi_multirec/build/rpi_multirec --hat-ui --mic spcmic
Restart=on-failure
RestartSec=2
User=root
Group=root
StandardOutput=journal
StandardError=journal

[Install]
WantedBy=multi-user.target
EOF

Step 6: Reload, enable, and test
sudo systemctl daemon-reload
sudo systemctl enable rpi_hat_boot_screen.service
sudo systemctl enable rpi_multirec.service
sudo systemctl restart rpi_hat_boot_screen.service
sudo systemctl restart rpi_multirec.service
sudo systemctl status rpi_hat_boot_screen.service --no-pager
sudo systemctl status rpi_multirec.service --no-pager

Step 7: Reboot validation
sudo reboot

Troubleshooting
If splash service fails:
journalctl -u rpi_hat_boot_screen.service -b --no-pager

If recorder does not start:
journalctl -u rpi_multirec.service -b --no-pager

If LCD module import fails:
PYTHONPATH=/home/hcenteno/rpi_multirec/python python3 -c "import ST7789; print('ok')"

Disable splash quickly:
sudo systemctl disable --now rpi_hat_boot_screen.service

Disable recorder autostart quickly:
sudo systemctl disable --now rpi_multirec.service
